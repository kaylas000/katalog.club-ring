# AI Boxing Coach — Бриф для разработки

## Статус: Идея → Планирование

## Суть проекта

Мобильное приложение высокого качества для **анализа, оценки и тренировки боксёрских приёмов** с использованием компьютерного зрения и ML.

Приложение должно быть **лучше** существующих решений:
- AI Boxing Coach (vlts1) — Flutter, базовый анализ
- BoxingWithML (iv4n-ga6l) — Python, простой подсчёт
- CompuBox (shangar21) — Python, профессиональный анализ

## Целевая аудитория

- Боксёры (от любителей до профессионалов)
- Тренеры (для анализа учеников)
- Фитнес-клубы (боксёрские секции)
- Фанаты бокса (для развлечения)

## Ключевые фичи

### 1. Запись видео
- Камера телефона (фронт/задняя)
- Разные углы (фронт, профиль, 45°)
- Запись приёмов (джеб, кросс, хук, апперкот, боковой)
- Автоопределение начала/конца удара

### 2. ML-анализ
- **Распознавание позы** — 17 точек тела (YOLOv8-pose)
- **Определение типа удара** — джеб, кросс, хук, апперкот (ActionNet)
- **Оценка попадания** — land/miss (HitNet)
- **Анализ техники** — углы, скорость, баланс

### 3. Оценка (0-100)
- **Техника** — правильность выполнения
- **Скорость** — скорость удара
- **Сила** — мощность удара
- **Баланс** — устойчивость
- **Защита** — положение рук после удара

### 4. AI-тренер
- Сравнение с эталоном
- Выявление ошибок
- Рекомендации по исправлению
- Прогресс-трекинг

### 5. Социальная составляющая
- Рейтинги
- Соревнования
- Обмен видео
- Комментарии тренеров

## Технологический стек

| Компонент | Технология | Обоснование |
|-----------|------------|-------------|
| **Приложение** | Flutter или Expo | Кроссплатформенность |
| **ML-ядро** | PyTorch + YOLOv8 | CompuBox + BoxingWithML |
| **Бэкенд** | Next.js API | Как на Club Ring |
| **БД** | PostgreSQL (Neon) | Как на Club Ring |
| **Хранилище видео** | Cloudflare R2 / S3 | Дёшево, быстро |
| **Авторизация** | Google OAuth + Email | Как на Club Ring |
| **Push-уведомления** | Firebase Cloud Messaging | Стандарт |

## Референсы (откуда брать примеры)

### Готовые решения
1. **AI Boxing Coach** — https://github.com/vlts1/ai-coach-demo
   - Flutter + C++ ML
   - Google Play / App Store
   - Демо: combos lesson

2. **BoxingWithML** — https://github.com/iv4n-ga6l/BoxingWithML
   - MediaPipe + OpenCV
   - Простой подсчёт ударов
   - Эмуляция клавиш

3. **CompuBox** — https://github.com/shangar21/compubox
   - YOLOv8-pose + ActionNet + HitNet
   - Профессиональный анализ
   - JSON-вывод

### ML-модели
- **YOLOv8-pose** — распознавание позы
- **ActionNet** — классификация ударов
- **HitNet** — определение попадания
- **MediaPipe Hands** — распознавание рук

### Библиотеки
- **PyTorch** — ML-фреймворк
- **OpenCV** — обработка видео
- **NumPy** — математика
- **SciPy** — статистика
