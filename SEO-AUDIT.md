# SEO Аудит Club-Ring

**Общая оценка: 28/100**

---

## КРИТИЧНЫЕ ПРОБЛЕМЫ

### 1. Нет robots.txt
Файл отсутствует. Добавить в `public/robots.txt`:
```
User-agent: *
Allow: /
Disallow: /shop/cart
Disallow: /shop/checkout
Disallow: /shop/confirmation

Sitemap: https://club-ring.ru/sitemap.xml
```

### 2. Нет sitemap.xml
Нет XML-карты. Создать `app/sitemap.ts` с генерацией URL для всех клубов, товаров, новостей.

### 3. Нет Schema.org (JSON-LD)
Не одна страница не использует структурированные данные. Нужны:
- `Organization` — главная
- `SportsActivityLocation` — каждая страница зала
- `Product` + `Offer` — каждый товар
- `Article` + `NewsArticle` — новости
- `BreadcrumbList` — везде где есть крошки
- `FAQPage` — энциклопедия

### 4. Клиентские страницы без мета-тегов
10 из 12 страниц используют `"use client"` и не экспортируют `metadata`. Google видит только глобальный title из layout. Конкретные страницы:

- `/clubs` — нет title, description
- `/clubs/[slug]` — нет title, description  
- `/shop` — нет title, description
- `/shop/[category]` — нет title, description
- `/shop/[category]/[slug]` — нет title, description
- `/trainer` — нет title, description

**Решение:** вынести мета-теги в отдельные server-компоненты или использовать `generateMetadata` в page-родителе.

### 5. Нет canonical тегов
Ни одна страница не указывает canonical URL. Дублирование контента при наличии query-параметров.

### 6. Нет Open Graph / Twitter Card тегов
Только глобальные мета-теги. Каждой странице нужны og:title, og:description, og:image, og:url.

---

## ВЫСОКИЙ ПРИОРИТЕТ

### 7. Контент дублируется в client-рендеринге
Главная страница, каталог залов, магазин — всё рендерится на клиенте. Поисковики могут не увидеть контент. Нужен SSR/SSG для ключевых страниц.

### 8. Изображения без атрибутов width/height
Все `<img>` теги (кроме trainer) не имеют размеров — это вызывает CLS (Cumulative Layout Shift).

### 9. Нет внутренних перелинкований между разделами
Новости не ссылаются на залы, залы не ссылаются на магазин. Слабая перелинковочная структура.

### 10. Нет hreflang
При расширении на другие языки будет проблема. Пока не критично.

---

## СРЕДНИЙ ПРИОРИТЕТ

### 11. H1 на каждой странице только один (хорошо)
Но на `/clubs/[slug]` H1 = название клуба — это правильно.

### 12. Breadcrumbs реализованы визуально, но без Schema.org
Крошки есть на страницах, но без микроразметки BreadcrumbList.

### 13. Ключевые слова в URL есть
/slugs чистые: `/clubs/boxing-club-atlant-moscow`, `/shop/gloves/cleto-reyes-pro-gloves`.

### 14. Нет мета-тега viewport
Нет явного viewport meta — Next.js добавляет по умолчанию, но лучше указать явно.

---

## НИЗКИЙ ПРИОРИТЕТ

### 15. Нет lang на страницах с контентом на русском
Указан `lang="ru"` в layout — это хорошо.

### 16. Нет alt текстов у изображений товаров
В каталоге изображения товаров используют `alt={product.name}` — достаточно.

### 17. Видео без обработки
Hero-видео автоматически воспроизводится. Нет ленивой загрузки видео.
